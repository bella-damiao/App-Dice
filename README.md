# 🎲 App Dice

This app simulates dice rolling and displays random dice images on the user interface.

### 👩‍💻 Technologies and Concepts used:

- Swift Programming Language
- UIKit Framework
- Outlets and Actions
- Interface Builder and Storyboards
- UIImageView
- Randomization
- Auto Layout
